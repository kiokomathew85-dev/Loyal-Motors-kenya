##  Project Overview

Loyal Motors Kenya is a professional web application designed to bridge the gap between car buyers and premium vehicle inventory. This project focuses on providing a seamless user experience, allowing potential clients to browse available stock, learn about import services, and contact the dealership directly.

### Purpose & Problem Solved
The goal was to create a digital showroom that establishes trust through a clean UI and accessible information. It solves the problem of information fragmentation in the Kenyan car market by providing a centralized hub for vehicle details, pricing, and service inquiries.
